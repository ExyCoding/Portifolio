p h1 a h6

parágrafo

<p> conteudo </p>

a tag <p></p> semânticamente é um parágrafo.

titulos

<h1></h1> até <h6></h6>

a tag <h1></h1> até <h6></h6> são semanticamente títulos e subtítulos, do <h1></h1> é maior até o <h6></h6> é menor.

formatadores

<b></b>

a tag <b></b> faz com que o conteúdo fique em negrito e semanticamente conteúdo não importante.

<strong></strong>

a tag <strong></strong> faz com que o conteúdo fique em negrito e semanticamente importante.

<i></i>

a tag <i></i> é usada para um termo técnico e ficará em itálico.

<em></em>

a tag <em></em> é usado para ênfase verbal e ficará em itálico.

<small></small>

a tag <small></small> faz com que o conteúdo fique menor.

<mark></mark>

a tag <mark></mark> faz com que o conteúdo fique com o texto marcado, semanticamente é pra ser marcado ou destacado.

<del></del>

a tag <del></del> faz com que o texto fica com linha rabiscada sobre ele.

<ins></ins>

a tag <ins></ins> faz com que o texto fica sublinhado.

<sub></sub>

a tag <sub></sub> faz com que o texto fica pequeno e na parte de baixo, é usado para supor elementos químicos.

<sup></sup>

a tag <sup></sup> faz com que o texto fica pequeno e na parte alta, é usado para notas de rodapé.

citar

<blockquote cite="site"></blockquote>

a tag <blockquote></blockquote> define uma seção que é citada de outra fonte.

<q></q>

a tag <q></q> é para citações curtas, o conteudo ficará entre aspas ao renderizar.

<abbr title="abreviação"></abbr>

a tag <abbr></abbr> é uma abreviação, ao passar o mouse por cima da palavra, a abreviação aparecerá.

<address></address>

a tag <address></address> para informações de contato.

<cite></cite>

a tag <cite></cite> semânticamente é para titulos de trabalhos criativos, é renderizado em italico.

<bdo></bdo>

a tag <bdo></bdo> é usado para substituir a direção do texto atual, tendo como atributo `dir` e valores rtl(direita para a esquerda) e ltr(esquerda para direita).
