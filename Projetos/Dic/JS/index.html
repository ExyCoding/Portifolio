variáveis

var Comida;

cria-se uma variável chamada Comida, é de escopo global.

let Comida;

cria-se uma variável chamada Comida, é de escopo local.

const Comida;

cria-se uma variável chamada Comida, é de escopo global e seu valor não pode ser alterado.

tipos de dados

var NumeroQualquer = "";

o tipo da variável NumeroQualquer é uma string, pois se tiver aspas, é string.

var X = true;

o tipo da variável X é boolean, pois se for true ou false, é boolean.

var y = false;

o tipo da variável Y é boolean, pois se for true ou false, é boolean.
