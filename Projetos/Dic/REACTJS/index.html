componentes props

componentes

function Hello(){
return(
<>
<main></main>
 </>
)
}

criamos um componente chamado Hello, e ela retorna tudo que estiver em return.

para compartilhar o componente, devemos adicionar outro codigo no final do arquivo JavaScript.

export default Hello ;

agora nosso componente pode ser compartilhado e ultilizado.

para adicionar o css do componente, devemos criar o arquivo css na mesma pasta com o mesmo nome do arquivo JavaScript mas mudar o tipo do arquivo de js para css, logo após isto devemos adicionar um codigo no começo do arquivo JavaScript.

import "./Comp.css" ;

para importar um componente é facil, no inicio do código de onde o componente será importado, colocaremos o seguinte.

import Hello from "./Hello" ;

agora importamos o componente Hello

props

<Hello Texto="Hello Word"/>

aqui, ultilizamos o componente Hello, e ele tem um atributo chamado Texto, ele é uma props.

function Hello(props){
return(
<>
<main> {props.Texto} </main>
 </>
)
}

o componente Hello recebe como argumento a prop, e o componente retornará o codigo descrito.
